## Quick Start Guide

### Prerequisites
- PHP 8.1+
- MySQL 8.0+
- Web server or PHP built-in server

### Setup (5 minutes)
1. **Extract the project files**
2. **Create MySQL database**: `pet_store`
3. **Import database**: Run `schema.sql` and `sample_data.sql`
4. **Start server**: `php -S localhost:8000` in the `public/` directory
5. **Access website**: Open `http://localhost:8000`
